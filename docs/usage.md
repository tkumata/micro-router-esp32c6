# 使用方法

## 1. マイクロルータの起動

1. XIAO ESP32C6 を USB HUB やモバイルバッテリなどに接続します
2. スマホなどの端末で「micro-router-esp32c6」 SSID に接続します
3. その際のパスワードは「esp32c6router」です (デフォルト)

## 2. Wi-Fi 認証情報の入力・保存

1. スマホなどの端末の設定から「micro-router-esp32c6」という AP に接続します
2. 接続すると、キャプティブポータル機能で「http://micro-router.local」へ自動的に遷移します
   このキャプティブポータル機能は、初回時のみです
3. 自宅の Wi-Fi AP の認証情報 (SSID とパスワード) を入力します
4. 「保存して再起動」ボタンをクリックします

## 3. 動作確認

1. 端末が「micro-router-esp32c6」 SSID に接続されていることを確認します
2. https://www.google.com にブラウザでアクセスします

## 4. AP パスワードの変更

セキュリティ向上のため、AP のパスワードをデフォルトから変更することを推奨します。

1. 「micro-router-esp32c6」に接続した状態で http://micro-router.local にアクセスします
2. Web UI の「AP パスワード変更」セクションを見つけます
3. 新しい AP パスワードを入力します (8〜64 文字)
4. 「AP パスワードを保存して再起動」ボタンをクリックします
5. デバイスが再起動するまで待ちます (約 5 秒)
6. 再起動後、新しいパスワードで「micro-router-esp32c6」に接続してください

**注意**: パスワードを忘れた場合は、デバイスの Flash メモリを消去してリセットする必要があります。
